# 🏎️ F1 Strategy Intelligence Engine

A machine learning-based Formula 1 race strategy simulator that predicts lap times, evaluates pit stop strategies, and recommends optimal race strategies through an interactive dashboard.

---

## 🚀 Features

- 🧠 ML-based lap time prediction using Random Forest
- 🏁 1-stop and 2-stop strategy optimization
- 🎲 Monte Carlo simulation for uncertainty analysis
- 📊 Strategy comparison and heatmaps
- 📉 Tire degradation visualization
- 🎛 Interactive Streamlit dashboard

---

## 🧠 How It Works

1. Historical F1 lap data is collected using FastF1  
2. A machine learning model predicts lap time  
3. Race simulation runs for different strategies  
4. Optimization finds best pit stop strategy  
5. Monte Carlo simulation models uncertainty
